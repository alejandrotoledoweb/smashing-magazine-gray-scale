PERSON OF THE WEEK

Álvaro Trigo (@IMAC2) is a self-taught web developer who aims to combine the beauty of design with the logical perfection of coding. Years ago, he built his first JavaScript plugin, with no experience and as an experiment to improve his skills; today, his code (the fullPage.js library, for example) is used worldwide by companies like Google, eBay, BBC, and Sony. Álvaro’s advice: “Don’t wait to be a master, start now!” Thank you, dear Álvaro, not only for providing developers with useful tools but also for inspiring everyone to start creating.
